Zen includes a small set of gradient images that are used by sample programs.
If you wish you can also use these images within your Zen applications (or at least until you get around to creating the real styles for your application).
All of these gradients were created using a nice on-line tool called Gradient Image Maker: http://tools.dynamicdrive.com/gradient/
The naming convention for these files is:
grad-(color)-(width)x(height).png
grad-(color)2-(width)x(height).png  inverted image
Typically you use these gradients as follows (with background color set to the end color of the gradient):
<style>
.myStyle {
background: #000040;
background-image: url(/csp/broker/images/grad-blue-10x100.png);
background-repeat: repeat-x;
}
</style>
